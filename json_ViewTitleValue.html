ViewTitleValue Data Type

Mediatri titles can have special format of the value data stored in DB. Server creates the separate object to present title value which is returned as Json model to the client for viewing. All title value should have presentation property. BaseTitleValue class contains default presentation implementation for the all titles. All title value classes should be derived from BaseTitleValue and reimplement buildPresentation() method if it is necessary some special text presentation of the value. Mediatri logic requires get vapaa or vakio fields from title in different use cases. CoreApi also need this functionality. Therefore two methods added: - getVapaa() - getVakio() By default these methods return empty String. Derived title value classes can redefined these emethods. By default these values are included into client Json model.
