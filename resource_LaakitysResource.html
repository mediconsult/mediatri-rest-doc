GET /medicationlist/interactions

Returns a list of interactions between medicines given in parameters

Request Parameters
name type description multivalued
id query List of medicines (VNR codes) yes
Response Body
media type data type description
application/json array of Yhteisvaikutus (JSON) List of interaction objects

Example

Request
GET /medicationlist/interactions
Accept: application/json
Response
HTTP/1.1 200 OK
Content-Type: application/json

                
[ {
  "laake1" : "...",
  "laake2" : "...",
  "interaktio" : "..."
} ]
