MainTitleConfig Data Type

DTO model to present configuration for main titles used in kertomus configuration DTO KertomusFormConfig. This configuration includes two lists:

  • Map of titles from Mallikirjaukset.

    Key is the otsikko name from teksti table and value is title's value stored in vapaa and vakio fields.

    Based class from ViewTitleValue presents the default value depending from type of otsikko.
  • Map of groups with list of titles in every group.
Main title can have general näkymä/view code protection. Codes under protection is hardcoded now and contains in core.properties file. Server reads core.properties file to check if main title is under propection.
