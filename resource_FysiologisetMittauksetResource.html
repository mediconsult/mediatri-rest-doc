GET /measurements/{id}

Fetch the list of measurements. Service returns the list of measurements for the last month from the current day. The measurements are grouped by title first. Titles are returned in alphabetical order.

Request Parameters
name type description constraints
id path Patient omistajaId for whom to fetch the data int
count query Set the limit to the result set. Default is 5 if no value is given. It means that service returns 5 latest measurements for each title. int
start query long
Response Codes
code condition
400 Bad Request: if values were invalid
401 Unauthenticated: The request requires user authentication.
403 Forbidden: User is authenticated but doesn't have enough priviledges to access this data. Re-login will not help and the request SHOULD NOT be repeated.
410 Gone: Patient has been removed
412 Precondition failed: Authenticated user must pick proper encounter event before accessing this API
500 Unexpected error
Response Body
media type data type description
application/json FysiologisetMittauksetView (JSON) list of measurements

Example

Request
GET /measurements/{id}
Accept: application/json
Response
HTTP/1.1 200 OK
