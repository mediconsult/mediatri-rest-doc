EventMissingResourcesDto Data Type

Data transfer object for missing resource information. Comparison is done against requested resources of calendar event related event model.

Properties
name data type description
numOfMissingWorkers number Amount of missing workers
missingSkill number Id of the missing worker skill if only one missing worker. Null if no or multiple missing workers.
numOfMissingEquipmentTypes number Amount of missing equipment types
missingEquipmentType number Id of the equipment type if only one type of equipment is missing. Null if no or multiple missing equipment types.
numOfMissingEquipment number Amount of the missing pieces of equipment if only one type of equipment is missing. Null if no or multiple missing equipment types.
numOfRequiredEquipment number Amount of the required pieces of equipment if only one type of equipment is missing. Null if no or multiple missing equipment types.
numOfMissingRoomGroups number Amount of missing rooms groups
missingRoomGroup number Id of the missing room group if only one missing. Null if no or multiple missing room groups.
