AdminResource Resource

REST API for various admin tasks like clearing cache onfly.

GET /admin/clearcache

Api for clearing cache. This ensures that all configuration is refetched from mediatri. It uses not obligatory request parameters

Request Parameters
name type description constraints
cacheid query int
Response Codes
code condition
400 Bad Request: if values were invalid
401 Unauthenticated: The request requires user authentication.
403 Forbidden: User is authenticated but doesn't have enough priviledges to access this data. Re-login will not help and the request SHOULD NOT be repeated.
500 Unexpected error
Response Body
media type data type description
application/json object (JSON) String json ok:true

GET /admin/l2content

Checks if l2 cache contains given entity type with given id. This service is used with cache evict tests

Request Parameters
name type description
entityType query full class name of the entity eg. fi.mediconsult.core.internal.entity.laakitys.LaakeValmisteJpa
id query Id of the entity in serialized string.
Response Codes
code condition
400 Bad Request: if values were invalid

500 Unexpected error

POST /admin/systemProperty

Set value for the system property. Please note, this set only for next Wildfly reboot, and this method is purposed only for test setup purposes Property needs to be defined as a changeable property, if not, Bad Request will be thrown.

Request Parameters
name type description
propKey query name of the system property
propValue query value of the system property
Response Codes
code condition
400 Bad Request: given property is not defined as a changeable
500 Unexpected error
Response Body
media type data type description
application/json string (JSON) previous value of the property or null if not existed before

Example

Request
POST /admin/systemProperty
Accept: application/json
Response
HTTP/1.1 201 Created
Content-Type: application/json
