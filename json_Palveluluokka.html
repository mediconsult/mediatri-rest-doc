Palveluluokka Data Type

Stores the information of the following ASETUS-table fields configured for Palveluluokkas: 0. Nimi (Yleiset tab) 1. Käyttäjäryhmä (Yleiset tab) 2. Palvelun tarve-otsikko (Yleiset tab) 3. Päätöstekstin otsikko (Yleiset tab) 4. Lakipykälät (Yleiset tab) 5. Suoritekoodi (parsed, but not added Core DTO) 6. Diaarinumerosarja (Yleiset tab) 7. Päätös avoin (K/E, Yleiset tab) 8. Riviotsikko (Maksuperusteet tab) 9. Sarakeotsikko (Maksuperusteet tab) 10. (not used currently in Mediatri) 11. Laskutus tapahtumien mukaisesti (K/E, Seuranta tab) 12. Tukipalvelu (K/E, Yleiset tab) 13. (a field related to Suorite-handling, parsed, but not added Core DTO)) 14. Päättäjäryhmän otsikko (Yleiset tab) 15. (not used currently in Mediatri) 16. Palvelunimikkeistö ((Yleiset tab) 17. Tapahtumamalli (Yleiset tab)
