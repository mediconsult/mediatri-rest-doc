AuthenticationResource Resource

Service for authentication

POST /authentication/login

Login with username and password. This returns with http-200 even authentication fails. Response has "success" true/false if authentication was successful.

Request Body
media type data type description
application/x-authc-username-password+json LoginCredentials (JSON) LoginCredentials with userId and Password.
Response Codes
code condition
500 Unexpected critical error.
Response Body
media type data type description
application/json CoreAuthenticationDTO (JSON) CoreAuthenticationDTO on successful login

POST /authentication/contextLogin

Login with context. Currently this will join context once with given sessionKey and fetch all keys. Context login will fetch User.Co.[mediconsult.fi]UserId that is mediatri omistajaId. It can be professional or customer too. Customers have very limited priviledges. If context doesn't contain value in User.Co.[mediconsult.fi]UserId then User.Id.Logon value is used to search for username from mediatri. User.Id.Logon login is limited only to professionals logging in. *

Request Parameters
name type description
domain form Mediatri-domainId
sessionKey form Context session-key to be used in join
Response Codes
code condition
500 Unexpected error
Response Body
media type data type description
application/json CoreAuthenticationDTO (JSON) 200 if ok

GET /authentication/login

API for authenticating to Core. Can be used in browser http://localhost:8080/CoreWEB/rest/authentication/login?username=MGR&password=&mediatritunnus=

Request Parameters
name type description constraints
mediatritunnus query Mediatri-domainID  
password query password for user. Password can be reset in mediatri  
role query Must have optionalUnitId-parameter too if role is given. If role+unit doesnt exist, bad request is thrown and user is logged out. int
unit query UnitId can be given directly as parameter-> session will use this unit and no 2-nd login step is required. int
username query Username eg "MGR"  
Response Codes
code condition
500 Unexpected error
Response Body
media type data type description
application/json CoreAuthenticationDTO (JSON) Status 200 with json message status:"Authenticated"
