LaakitysKayttoaihe Data Type

DTO for LAAKITYS_KAYTTOAIHE and KAYTTOAIHE database tables http://vc-buildsrv1/documents/mediatridocs/latest/tables/LAAKITYS_KAYTTOAIHE.html http://vc-buildsrv1/documents/mediatridocs/latest/tables/KAYTTOAIHE.html

Properties
name data type description
rivi number Returns the row number, which arranges the LaakitysKayttoaihe entities in order of importance
numero number Returns the unique id of Kayttoaihe
selite string Returns the name of the Kayttoaihe in plain languate
reseptiteksti string Returns the name of the Kayttoaihe which is printed to the prescription
alkupera object Returns the origin of the Kayttoaihe -"O" if added by organization itself -"A" or "M" otherwise
tyyppi object Returns the type of the Kayttoaihe -"C" if Kayttoaihe -"G" if group -"D" if deleted
poistettu object Returns the code of the kayttoaihe removal. 'E' - not removed, 'K' = removed
muutosaika number Returns the actual DB-change time of the Kayttoaihe
muutospaikka number Returns the organization in which the change has been made
