TitleResource Resource

REST API for title resource. Supported operations: GET titles/{title name}/config returns configuration for the titles titles/{title name}/options returns list of available values for the title

GET /titles/mtconfigs

Return main titles' configurations by list of main titles as parameter of the request. List is returned in alphabetical order. The list includes only main titles which configured in the system and contain the configured titles.

Request Parameters
name type description multivalued
titles query list of main titles. It can be null. In this case all main titles will be returned. yes
Response Codes
code condition
400 Bad Request: if values were invalid
401 Unauthenticated: The request requires user authentication.
403 Forbidden: User is authenticated but doesn't have enough priviledges to access this data. Re-login will not help and the request SHOULD NOT be repeated.
500 Unexpected error
Response Body
media type data type description
application/json array of MainTitleConfig (JSON) main titles titles' configurations by component

Example

Request
GET /titles/mtconfigs
Accept: application/json
Response
HTTP/1.1 200 OK
