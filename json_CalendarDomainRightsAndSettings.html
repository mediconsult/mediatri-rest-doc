CalendarDomainRightsAndSettings Data Type

Data transfer object for calendarForm domain rights and settings.

Properties
name data type description
name string Get the name of the calendarForm.
setupForm FormRights Get information about the Mediatri form and current user rights for viewing and modifying setupForm data: reporting groups, event models, rooms, room groups and equipment.
patientList FormRights Get information about the Mediatri form and current user rights for the patient list.
patientLists array of FormRights
patientSearchAllowed boolean
workerLists array of FormRights Get the list of Mediatri forms and current user rights for the worker lists.
workerSearchAllowed boolean
calendarForm FormRights Get Mediatri form information and current user rights for the calendar, i.e. viewing and modifying events.
coursesForm FormRights Get Mediatri form information and current user rights for courses, i.e. viewing and modifying courses, course periods and course related settings (tags, named groups, roles).
coursesUsed boolean
courseSettings CourseSettings
workerSettingsForm FormRights Get Mediatri form information and current user rights for worker related information, i.e. viewing and modifying worker skills and working hours.
optimization OptimizationRightsAndSettings Get Mediatri form information and current user rights and settings for optimization.
primaryNurseSettings PrimaryNurseSettings
viewTitle string
