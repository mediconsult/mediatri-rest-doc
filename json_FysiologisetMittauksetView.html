FysiologisetMittauksetView Data Type

Basic DTO model for physiological measurements. This class includes list of measurements objects which are the instances of FysiologinenMittaus. MittausComponentConfig class presents the configuration used in adding new measurements. This configuration includes the list of kertomusteksti forms and main titles configured for every form.

Properties
name data type description
patientId number
fysiologinenMittausList array of FysiologinenMittaus
