PalvelutapahtumaList Data Type

This class is dto for transferring multiple different palvelutapahtumas in single query. This has:

  • list of palvelutapahtumas
  • reference to preferred palvelutapahtuma from list (Care relation requires this)

Properties

allItemsCount number
entries array of Palvelutapahtuma
preferredPalvelutapahtumaReference array of number
preferredPalvelutapahtumaSelectStatus PalvelutapahtumaSelectType
