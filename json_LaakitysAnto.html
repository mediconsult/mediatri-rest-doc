Properties
name data type description
antaja number Returns the id of the medication giver
huomautuksia string Returns the observetion comments related to the medication giving
kirjaaja number Returns the id of the actual person that fed this info to the DB/system
laake string Returns the laake VNR. VNR is a code that is used to identify the medication. @see http://www.laaketietokeskus.fi/vnr
maara number Returns the medication amount
muutosAika number Returns the actual DB-change time of the medication giving
omistaja number Returns the owner ie. patients mediatri id.
osa number Retuns the part number that is the id of the medication together with alku and omistaja
poistettu object Returns the code of the medication time removal. 'E' - not removed, 'K' = removed
suunniteltuAika number Returns the planned time for medication. Tells which time the medication should be given
toteutunutAika number Returns the actual time of medication. Mapped from PK id ALKU
toteutunutAikaLoppu number Returns the actual end time of medication. a.k.a. LOPPU
tyyppi number Returns the type of LaakitysAnto object, 1 = planned and given, 2 = planned but not yet given, 3 = extra
yksikko string Returns the used unit. ml, l, etc.
kirjaajaName string Returns the name of the actual person that fed this info to the DB/system
antajaName string Returns the name of the medication giver
annettavaMaara number Returns the base amount that is given if swiped. Also the lower border of the amount.
annettavaYksikko string Returns the base unit that is described by laakityshistoria.
annettavaMaksimi number Returns the uuper border of the amount.
toteutunutAikaOriginal number Original time of medication. mapped from PK id ALKU. This PK is modified so to find the modifiable row the old row must be also given.
kello number
