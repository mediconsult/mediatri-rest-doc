TimeRuleDto Data Type

Data transfer object for defining working hour rules.

Properties
name data type description
start number Get the start time of the rule in Unix time
durationInMinutes number Get the duration of the rule in minutes
recurrenceEnd number Get the end time of a recurring rule in Unix time
recurrenceCode string Get the recurrence rule. The string is in a HL7 specified format. For example: Q6H indicates that the event is performed at least once every 6 hours around the clock. QJ1 indicates that the event is performed at least every week on Mondays. QAM~QPM indicates that the event is performed at least once every morning and every evening. QJ1~QJ3~QJ5 indicates that the event is performed at least every week on Mondays, Wednesdays, and Fridays. C indicates that the event is performed continuously, 7 days per week.
