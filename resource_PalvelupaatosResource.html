GET /patients/{id}/serviceDecisions

Gets service decisions for given patient, with given time interval restrictions..

Request Parameters
name type description constraints
id path patient id int
endTime query end time (yyyy-MM-dd) for narrowing the returned result set (optional). Also more detailed timestamp allowed (yyyy-MM-dd'T'HH:mm:ss), but decisions are timestamped only at day-level.  
onlyValid query if set to true, returns only valid palvelupäätökses (optional) i.e. the ones matching criteria (POISTETTU=E && TILA=V && PAATOSLAATU=M || TILA=K && PAATOSLAATU=NULL || TILA=K && PAATOSLAATU=M) boolean
serviceClass query if set with non-empty/valid value (i.e. configured/recognized valid service class abbreviation), returns only service decisions matching given service class (optional)  
startTime query start time (yyyy-MM-dd) for narrowing the returned result set (optional). Also more detailed timestamp allowed (yyyy-MM-dd'T'HH:mm:ss), but decisions are timestamped only at day-level.  
Response Codes
code condition
400 Bad Request: if values were invalid
401 Unauthenticated: The request requires user authentication.
403 Forbidden: User is authenticated but doesn't have enough priviledges to access this data. Re-login will not help and the request SHOULD NOT be repeated.
500 Unexpected error
Response Body
media type data type description
application/json array of Palvelupaatos (JSON) list of service decisions
