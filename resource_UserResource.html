GET /users/search

Makes search from all users with given criteria. Either none or all search parameters can be set.

Request Parameters
name type description default constraints multivalued
endTime query availability condition (user account validity end time, inclusive), or null if not used as criterion   long no
firstNamePrefix query first name starts with condition, or null if not used as criterion     no
fullNameContains query user's full name contains string condition, or null if not used as criterion Full name search overrides more specific lastNamePrefix, firstNamePrefix, userNamePrefix searches.     no
lastNamePrefix query last name starts with condition, or null if not used as criterion     no
list query optional user list id(s), if not present users are retrieved from global register   int yes
pageIndex query page index starting from which client wants to list users 1 int no
pageSize query defines page size i.e. maximum amount of users returned in on query 50 int no
skill query skill(s) supported condition, or an empty list if not used as criterion   long yes
startTime query availability condition (user account validity start time, inclusive), or null if not used as criterion   long no
unit query optional unit ids, if present users lists for given units are retrieved, otherwise login unit is assumed   int yes
userNamePrefix query username starts with condition, or null if not used as criterion     no

code condition
