UnitResource Resource

REST API for units.

GET /units/list

Gets list of units. If no limiting parameters are given, returns all active units from the system.

Request Parameters
name type description constraints
pageIndex query page index, starting from which client wants to list units. Minimum value for page index is 1. If pageIndex is set, pageSize must also be given. int
pageSize query defines page size i.e. maximum amount of units returned in on query. If not set, returning all the units matching given conditions. int
Response Codes
code condition
400 Bad Request: if values were invalid
401 Unauthenticated: The request requires user authentication.
403 Forbidden: User is authenticated but doesn't have enough priviledges to access this data. Re-login will not help and the request SHOULD NOT be repeated.
500 Unexpected error
Response Body
media type data type description
application/json UnitList (JSON) unit list with paging related metadata

Example

Request
GET /units/list
Accept: application/json
Response
HTTP/1.1 200 OK
