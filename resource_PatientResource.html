GET /patients/list

Gets list of patients. If no limiting parameters are given, returns all active patients from the system.

Request Parameters
name type description constraints
maxId query optional parameter, sets minimum owner id for the returned patient list (not inclusive) int
minId query optional parameter, sets minimum owner id for the returned patient list (inclusive) int
pageIndex query page index, starting from which client wants to list patients. Minimum value for page index is 1. int
pageSize query defines page size i.e. maximum amount of patients returned in on query, if page size is set, then also pageIndex parameter must be defined. If not set, returning all the patients matching given conditions. int
Response Body
media type data type description
application/json PatientList (JSON) patient list with paging related metadata

GET /patients/search

Makes search from all patients with given criteria. Either none or all search parameters can be set.

Request Parameters
name type description constraints
firstNamePrefix query first name starts with condition, or null if not used as criterion  
fullNameContains query patient's full name contains string condition, or null if not used as criterion Full name search overrides more specific lastNamePrefix, firstNamePrefix, userNamePrefix searches.  
lastNamePrefix query last name starts with condition, or null if not used as criterion  
pageIndex query page index starting from which client wants to list patients int
pageSize query defines page size i.e. maximum amount of patients returned in on query, if page size is set, then also pageIndex parameter must be defined. If not set, returning all the patients matching given conditions. int
ssnPrefix query social security starts with condition, or null if not used as criterion  
Response Body
media type data type description
application/json PatientList (JSON) patients found with given criteria
