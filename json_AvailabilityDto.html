AvailabilityDto Data Type

Data transfer object for workers' availability information.

Properties
name data type description
start number Required start time in Unix time
end number Required end time in Unix time
available boolean Is the worker available to work in required time?
workingHoursPlanned boolean Does the worker have working hours planned in required time?
conflicts boolean Does the worker have conflicting events planned at required time?
workingHourEvents array of CalendarEventDto Get the working hour event(s) that cover the required time. If working time is not (fully) planned for the required time period, returns working hour events partially overlapping the period, or closest to the period if any exist during a 12 hour interval before of after the required time.
conflictingEvents array of CalendarEventDto Get any events that conflict with the required time.
