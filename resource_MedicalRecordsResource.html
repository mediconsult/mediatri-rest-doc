PUT /medicalreports/lock

Lock medical report before editing.

Request Body
media type data type description
application/json KertomusID (JSON) reportId to lock
Response Codes
code condition
401 Unauthenticated: The request requires user authentication.
403 Forbidden: User is authenticated but doesn't have enough priviledges to access this data. Re-login will not help and the request SHOULD NOT be repeated.
404 Not found: Patient or form is not found
423 Form is locked already
500 Unexpected error
Response Body
media type data type description
application/json number (JSON) lock date if success

Example

Request
PUT /medicalreports/lock
Content-Type: application/json
Accept: application/json

                
{
  "ownerId" : 12345,
  "aika" : 12345,
  "formId" : 12345,
  "reportId" : 12345
}
Response
HTTP/1.1 204 No Content
Content-Type: application/json

GET /medicalreports/{id}

name type description constraints multivalued
id path omistaja id int no

pageSize query page size. Can be ommitted. Default value is 20. int no
