GET /seuranta/{id}

Return mittaus view for seuranta component. It can return the list of all mittaus or only for one lomake

Request Parameters
name type description constraints
id path patient Id int
count query count of mittaus for one otsikko,not obligatory parameter by default 5 int
form query seuranta lomake id. If defined the list contains the value only for one seuranta int
Response Codes
code condition
400 Bad Request: if values were invalid
401 Unauthenticated: The request requires user authentication.
403 Forbidden: User is authenticated but doesn't have enough priviledges to access this data. Re-login will not help and the request SHOULD NOT be repeated.
410 Gone: Patient has been removed
412 Precondition failed: Authenticated user must pick proper encounter event before accessing this API
500 Unexpected error
Response Body
media type data type description
application/json FysiologisetMittauksetView (JSON) the list of all mittaus or only for one lomake
