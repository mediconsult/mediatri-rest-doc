ComponentRightDTO Data Type

ComponentRightDTO for transfering visible DTO to client componentID = component in question (enum) visible = is component visible on client side. This is normally the read access, but might be different in special cases PermissionDTO = all permissions. Thise can be used on client as right changes during session are not supported

Properties
name data type description
serialversionuid number
component Component Get the attached component
inDevelopment boolean
requiresTarget boolean Can the component be showed only if omistaja/target is selected
permissionDTO PermissionDTOOfExtendFormInfo Get the permissions for the bound component

Example

{
  "serialversionuid" : 12345,
  "component" : "PERINTA",
  "inDevelopment" : true,
  "requiresTarget" : true,
  "permissionDTO" : {
    "add" : true,
    "data" : {
      "lomakeId" : 12345,
      "name" : "...",
      "hoitosuhdevaatimus" : true,
      "vastuuyksikko" : 12345,
      "pohjavari" : 12345,
      "fullFormTypeId" : 12345,
      "formTypeInfo" : { },
      "ownerTypeInfo" : { },
      "formOwnerId" : 12345
    },
    "type" : "...",
    "change" : true,
    "view" : true,
    "remove" : true
  }
}
