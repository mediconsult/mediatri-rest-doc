PUT /labs/{id}/acknowledge

Acknowledge one or more laboratory results. If tutkimuspyyntoDTO is not valid, ValidationException is thrown. If tutkimuspyyntos in tutkimuspyyntoDTO are ordered by someone else than the current user, AccessDeniedException is thrown.

Request Parameters
name type description constraints
id path Patient id int
Request Body
media type data type description
application/json Tutkimuspyynto (JSON) tutkimuspyyntoDTO
application/xml tutkimuspyynto (XML)
Response Body
media type data type description
application/json object (JSON) response
